# Kvasir Kafka

Kafka [event] source for Kvasir.

## Installation

The package can be installed
by adding `kvasir_kafka` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:kvasir_kafka, "~> 1.0"}
  ]
end
```

The docs can
be found at [https://hexdocs.pm/kvasir_kafka](https://hexdocs.pm/kvasir_kafka).
